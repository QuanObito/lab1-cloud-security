📝 Nội Dung Nội Dung Thực Hành

Trang web tĩnh này được triển khai nhằm mục đích hoàn thành nội dung thực hành Tuần 1, bao gồm các mục tiêu cốt lõi sau:

  • Yêu cầu 1: Tìm hiểu lý thuyết cơ bản về kiến trúc Điện toán đám mây (IaaS, PaaS, SaaS), tìm hiểu về cơ chế hoạt động, vai trò đối với hiệu năng và an toàn của CDN (Content Delivery Network).
  • Yêu cầu 2: Xây dựng mã nguồn trang web tĩnh (HTML/CSS), đồng bộ hóa và quản lý mã nguồn thông qua hệ thống kiểm soát phiên bản GitHub, sau đó thực hiện cấu hình tự động triển khai (CI/CD) lên nền tảng Cloudflare Pages.
  • Yêu cầu 3: Tiến hành thực nghiệm, phân tích cấu hình an toàn bảo mật HTTP Response Headers qua công cụ Security Headers và đánh giá tối ưu hóa trải nghiệm, tốc độ tải trang qua Google PageSpeed Insights.

⚙️ Mô Hình Kiến Trúc Ứng Dụng

Hệ thống được cấu hình vận hành hoàn toàn tự động theo quy trình hiện đại:

Mã nguồn (Local) → Đẩy lên GitHub Repository → Kích hoạt Webhook → Cloudflare Pages tự động biên dịch và phân phối ra mạng lưới CDN toàn cầu.
